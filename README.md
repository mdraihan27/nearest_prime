# nearest_prime
This program was created with QT framework, open source version.
These project files will run only in QT.
Simply install QT framework on your computer. Then go to QT, and in QT open the file that has the .pro extension.
This program finds the nearest prime number of any given number. 
If there are two numbers that have the same distance from the given number, it shows both as nearest prime number. 
If the given number itself is a prime number, it shows that number as its nearest prime number.
